# learnopenai

A teaching workspace for mastering the OpenAI platform as an AI Product Engineer.

## Start here

1. Read [MISSION.md](./MISSION.md)
2. Open [Lesson 1 — Your First Responses API Call](./lessons/0001-first-responses-api-call.html)
3. Keep [Responses API Basics](./reference/responses-api-basics.html) handy while coding

## Structure

| Path | Purpose |
|------|---------|
| `lessons/` | Self-contained HTML lessons |
| `reference/` | Cheat sheets and quick-reference docs |
| `practice/` | Runnable scripts for hands-on exercises |
| `assets/` | Shared styles and interactive components |
| `learning-records/` | What you've learned (drives what comes next) |

## Prerequisites

- [platform.openai.com](https://platform.openai.com) account with API billing (~$5 minimum)
- Node.js 18+ for practice scripts
- Copy `.env.example` to `.env` and set your `OPENAI_API_KEY`
