# Research assistant as capstone project

Sam wants to assemble everything learned into a **research assistant** — a Next.js app for web-grounded Q&A with citations, multi-turn threads, and optional doc upload. Triage capstone patterns transfer directly; the domain shifts from internal backlog lookup to open-web research.

**Evidence:** User completed Lesson 24 and explicitly asked for incremental lessons that build the full app.

**Implications:** Lesson arc 25+ scaffolds `research-assistant/` and adds one vertical slice per lesson (scaffold → API route → streaming UI → structured brief → threads → file search → hardening). Cross-provider comparison (mission criterion #4) deferred until the app ships.
