Image Generation

Text APIs answer questions. Image APIs ship pixels.

Lesson 23 finished the capstone conversation loop. Image generation is the last core OpenAI primitive on your curriculum list — and it uses a different mental model than chat tokens. You pay per image (quality × size), store binary output yourself, and choose between two APIs depending on whether the UX is one-shot or conversational.

Image API — your first win

One prompt in, one PNG out. Pick a GPT Image model (gpt-image-1-mini is the budget learner default; gpt-image-2 is newest). GPT Image models return b64_json by default — save the bytes to your own storage. Legacy DALL·E models accept response_format: "url" instead. 

const result = await client.images.generate({
  model: "gpt-image-1-mini",
  prompt: "Minimal flat icon, green checkmark badge, white background",
  size: "1024x1024",
  quality: "low",
});

const b64 = result.data[0].b64_json;
fs.writeFileSync("badge.png", Buffer.from(b64, "base64"));
GPT Image may need org verification.
If you get a verification error, complete API organization verification in the OpenAI dashboard — or set IMAGE_MODEL in .env to a model your account can access. Image calls are billed separately from chat tokens; start with quality: "low".

Responses API — built-in tool

Same pattern as Lesson 8 (web_search, file_search): register a built-in tool, read platform output items. The image lands in an image_generation_call — not output_text. 

const response = await client.responses.create({
  model: "gpt-5-mini",
  input: "Generate a friendly green otter mascot in a hard hat",
  tools: [{ type: "image_generation" }],
  store: true,
});

const call = response.output.find((o) => o.type === "image_generation_call");
fs.writeFileSync("mascot.png", Buffer.from(call.result, "base64"));
// call.revised_prompt — mainline model may rewrite your prompt

Multi-turn edits reuse conversation state from Lesson 10 / 23 — chain with previous_response_id and a new instruction (“make it watercolor”).

Pricing

quality is the biggest cost lever — low → high is roughly 7–35×. Your practice script at gpt-image-1-mini + low is about half a cent per image. Full pricing table: image generation reference → Pricing.

Product decisions

Decision Recommendation
Where images live Your object storage (S3, R2, Vercel Blob) — not the client bundle
Progressive UI Stream with partial_images (costs extra per partial)
Route design Same server boundary as triage — API key stays server-side
When to skip OpenAI High-volume asset pipelines → compare Fal, Replicate, Stability
