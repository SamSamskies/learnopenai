Built-in Tools

Platform-managed tools — OpenAI runs the handler.

Tool type What it does Setup
web_search Live web lookup + citations Add to tools; model + per-search cost
file_search RAG over your uploaded files Vector store + vector_store_ids
code_interpreter Run Python in a sandbox Add to tools; container billing
mcp Remote MCP server tools server_url, approval policy
vs custom function Custom (type: "function") Built-in
Who executes Your server OpenAI platform
Typical turns Two — call then function_call_output One — result in same response
Output item function_call web_search_call, etc.
When to use Your DB, auth, business logic Generic capability (search, files, code) 
// Web search — single request
const response = await client.responses.create({
  model: "gpt-5",
  input: "What changed in the latest OpenAI changelog?",
  tools: [{ type: "web_search", search_context_size: "low" }],
});

response.output_text;  // answer with inline citations
// response.output includes web_search_call + message
Web search output Where
web_search_call output[] — search action, queries, status
message output[] — output_text + annotations
url_citation Per-URL metadata — must be visible in product UI
