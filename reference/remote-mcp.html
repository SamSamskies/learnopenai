External tool servers via type: "mcp" — Responses API.

Fact Detail
What it does OpenAI lists tools from a remote MCP server and calls them during a response
Declare tools: [{ type: "mcp", server_url, server_label, … }]
Auth Often authorization: "Bearer …" on the tool object
Output items mcp_list_tools, mcp_call, then message
Approvals require_approval: "always" | "never" | { tool_name: "never" }
Latency tip Keep mcp_list_tools in context — avoid re-listing each turn
Safety Only trust servers you control or verify — context exfiltration risk
