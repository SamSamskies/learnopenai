Rules of thumb

  • Behavior frozen. Restyle only — no changes to useChat, phases, or API routes.
  • Variables first. Add tokens to :root and @theme inline; avoid hard-coded hex in components.
  • Borders over shadows. Depth via tonal surfaces and 1px outlines.
  • Serif for content, sans for chrome. Brief headlines and empty state use font-serif; UI controls use Geist.
  • QA all phases. Empty → searching → streaming → done → error, plus upload chips and copy-brief.
