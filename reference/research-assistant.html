Research Assistant

Next.js capstone — web search, citations, threads, optional docs.

V1 product shape

A solo builder types a research question. The app searches the web, streams an answer with citations, and remembers context for follow-ups. Optional: upload a PDF or markdown file for grounded answers alongside web results.

User action Server does UI shows
Ask “What changed in Next.js 15?” web_search built-in tool Searching → streaming answer → citation links
Follow up “How does that affect App Router?” previous_response_id from last turn Same thread, new answer
Upload product spec file_search on vector store Answer cites doc + web sources

Citation rendering

After web_search, check message.content[0].annotations for url_citation entries — same pattern as practice/0008-web-search.mjs. Render title + link in the UI; never trust the model's inline URLs without matching annotations.
