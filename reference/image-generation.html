Image Generation

Image API for single shots · Responses image_generation for conversational edits.

Choose the API

Need API Call
One image from a prompt Image API client.images.generate()
Chat UI, multi-turn edits Responses API tools: [{ type: "image_generation" }]
Edit with mask or reference images Image API client.images.edit()

Image API — generate

const result = await client.images.generate({
  model: "gpt-image-1-mini",   // or gpt-image-2
  prompt: "Flat icon, green checkmark badge",
  size: "1024x1024",
  quality: "low",                // low | medium | high
});

const b64 = result.data[0].b64_json;  // default for GPT Image
fs.writeFileSync("out.png", Buffer.from(b64, "base64"));

Responses API — built-in tool

const response = await client.responses.create({
  model: "gpt-5-mini",         // gpt-5+ supports image_generation
  input: "Generate a gray tabby cat hugging an otter",
  tools: [{ type: "image_generation" }],
  store: true,
});

const call = response.output.find((o) => o.type === "image_generation_call");
const b64 = call?.result;
// call.revised_prompt — model may rewrite your prompt

Multi-turn edit (Responses)

const followUp = await client.responses.create({
  model: "gpt-5-mini",
  previous_response_id: response.id,
  input: "Make it look like a watercolor painting",
  tools: [{ type: "image_generation" }],
});

Tool action: auto (default), generate, or edit. See image generation guide.

Output shape

API Where the image lives
Image API result.data[n].b64_json or .url
Responses API output[] item type: "image_generation_call" → result (base64)
Responses stream response.image_generation_call.partial_image events → partial_image_b64

Pricing

Image billing is image output tokens (plus text/image input on edits) — not chat completion pricing. OpenAI publishes per-image estimates at standard sizes; see cost and latency and the pricing page.

Per image — 1024×1024 square

Model Low Medium High
gpt-image-1-mini $0.005 $0.011 $0.036
gpt-image-1 $0.011 $0.042 $0.167
gpt-image-1.5 $0.009 $0.034 $0.133
gpt-image-2 $0.006 $0.053 $0.211

Portrait (1024×1536) and landscape (1536×1024) differ slightly — often a bit cheaper at the same quality. quality is the biggest lever (low → high is roughly 7–35× depending on model).

Extra charges

Factor Effect
Responses API Mainline model tokens plus image tool cost (e.g. gpt-5-mini orchestration)
Edits / reference images Image input tokens on top of output
partial_images streaming ~100 image output tokens per partial preview
n > 1 Multiplies image output cost
Budget defaults for iteration: gpt-image-1-mini + quality: "low" via Image API. Use Responses image_generation only when you need multi-turn edits.

Cost Per-image table — quality and size drive price; Responses adds chat tokens.
