File Search

Hosted RAG — vector store + file_search built-in tool.

Setup pipeline

files.create(purpose: "assistants")
  → vectorStores.create()
  → vectorStores.files.create(vector_store_id, file_id)
  → poll until status === "completed"
  → responses.create({ tools: [{ type: "file_search", vector_store_ids }] })
Step API Notes
Upload client.files.create purpose: "assistants"
Store client.vectorStores.create Reusable knowledge base id
Attach vectorStores.files.create Poll until indexed
Query responses.create + file_search One request — no turn 2

Tool config

tools: [{
  type: "file_search",
  vector_store_ids: ["vs_…"],
  max_num_results: 3,        // fewer = cheaper/faster
  filters: { … },            // metadata on vector store files
}]

Response shape

Output item Contains
file_search_call status, queries used for retrieval
message output_text + file_citation annotations 
const call = response.output.find(i => i.type === "file_search_call");
const msg = response.output.find(i => i.type === "message");

call?.queries;
response.output_text;
msg?.content[0].annotations;  // [{ type: "file_citation", filename, file_id, index }]
