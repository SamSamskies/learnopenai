Responses API

The unified OpenAI platform surface — text, tools, images, state.

Field / call What it is
POST /v1/responses Create a model response. Replaces most Chat Completions + Assistants patterns.
input String or array of messages/items sent to the model.
model Model ID, e.g. gpt-4.1-nano, gpt-4o-mini, o4-mini.
output Array of output items: messages, tool calls, reasoning, etc.
output_text SDK shortcut — aggregated text from all output_text parts.
id resp_… — pass as previous_response_id for multi-turn state.
usage input_tokens, output_tokens, total_tokens.
tools Built-in (web search, file search) or custom function definitions.
store When true (default), OpenAI stores the response server-side. 
import OpenAI from "openai";

const client = new OpenAI(); // reads OPENAI_API_KEY

const response = await client.responses.create({
  model: "gpt-4.1-nano",
  input: "Your prompt here",
});

console.log(response.output_text);

Mental model

Request → input + model (+ optional tools, instructions)
Response → output[] items (not a flat string)
Read text → response.output_text in JS/Python SDKs
Continue thread → previous_response_id: response.id
